AI-Powered Multilingual Healthcare Chatbot

Production medical NLU system for rural Indian healthcare — built with a strict zero-cloud hot-path guarantee. Every request flows through an 8-stage local pipeline: SemanticGate (spaCy + BioBERT) → custom NER (trainsem) → SemanticNormalizer (ICD codes, synonym expansion) → TriageModule (severity scoring) → RedFlagDetector (emergency bypass) → Interpreter → LoRA-fine-tuned Mistral-7B (checkpoint-1869) → ResponseRephraser. Red-flag conditions never reach the LLM — they short-circuit to immediate emergency referral. Bilingual Hindi–English via Bhashini API at presentation layer only. ASR pipeline for voice input. 22+ medical specialties with Hindi display names. Dockerized with Gunicorn for production deployment.
